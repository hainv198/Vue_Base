    • JOB DESCRIPTION

    - Install, configure & maintain computer hardware operating system (server, computer, laptop, printer, etc.)
    - In charge of security of the entire system, especially internal data storage, backup data management
    - Troubleshoot system and network problems, diagnose and resolve hardware or software failures, install anti-virus protection, fix network errors...
    - Administration and monitoring monitor daily network usage, technical support for network users
    - Internet system administration, Wifi system (Controller & Access Point), office equipment
    - Planning future improvements, Recommend IT upgrade solutions to Manager
    - Provide support, including other technical and documentation support
    - Other duties as assigned by Manager

    JOB REQUIREMENTS

    - Gender: Male
    - Being able to work immediately is an advantage.
    - Working in shifts
    - University Degree in IT and/or Telecommunications
    - 2+ years of experience in IT supporting Hardware / Software / Network and fluent in all IT fields
    - Having experience or knowledge about Microsoft Windows Server 2012, 2016 System (best if having MCSA, MCSE certificate).
    - Experience or knowledge in Cisco, Fortinet networking products such as Router, Switch, Wifi Controller (best if CCNA certificate is available).
    - Understood most of the following OS: Windows or Red Hat server including hands-on experience in an enterprise environment serving customers. Knowledge of firewalls, load balancing, WAN technologies, software-defined networking/virtualization, unified networking, multi-context network devices, and distributed application controllers.
    - Experience in customer service
    - Basic communication English
    - Ability to explain technical problems in a simple, team-oriented manner, flexible and reliable attitude
